I'm Alicia Wood, a developer based in South Australia. When I'm not coding, I can be found still huddled over a computer writing novels, or getting some rare sunshine with my dogs.

    • Profiency in 'HyperText Markup Language' or 'HTML', which is the standard markup language for documents designed to be displayed in a web browser.

    • Proficiency in 'Sass' which is a CSS preprocessor that adds special features such as variables, nested rules and mixins into regular CSS.

    • JavaScript is a programming language that is one of the core technologies of the World Wide Web (alongside HTML and CSS), and can be used to create dynamic and interactive web content.
